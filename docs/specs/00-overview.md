# CCZen 规格总览（Spec 00 — Overview)

> 状态：Draft v0.1 · 适用平台：Windows 10 (1809+) / Windows 11 · 语言：spec 阶段不含代码

## 1. 产品定位

CCZen 是一款**通用型 Windows 磁盘清理软件**，目标是在任意用户、任意软件环境的 Win10+ 电脑上都能获得良好的清理效果：

- **秒级扫描**：像 Everything 一样，秒级找出全盘大文件、大文件夹（核心卖点）
- **智能推荐**：基于本地规则引擎 + 多信号启发式评分给出"推荐清理"，不依赖任何 LLM，不联网也可完整工作
- **通用而非写死**：清理目标通过"环境发现 → 证据收集 → 置信度评分"推导，而不是硬编码路径；同时对主流应用（微信、浏览器、开发工具、游戏平台等）提供声明式特殊适配
- **绝对安全**：分级风险模型、隔离区可撤销删除、保护清单，宁可少清不可误删

## 2. 用户与场景

| 用户 | 场景 |
|------|------|
| 普通用户 | C 盘爆红，想一键安全释放空间 |
| 进阶用户 | 想知道"是什么占了我的盘"，按大小/类型/时间钻取 |
| 开发者 | node_modules、包管理器缓存、Docker vhdx、构建产物泛滥 |
| 游戏玩家 | 着色器缓存、旧安装包、更新残留 |
| IT 运维 | 批量/静默模式（远期） |

## 4. 关键设计决策（ADR 摘要）

| # | 决策 | 理由 |
|---|------|------|
| D1 | NTFS 卷直接枚举 MFT（`FSCTL_ENUM_USN_DATA`）建立全卷内存索引 | Everything 秒搜的同款原理；遍历目录树 API 慢 1~2 个数量级 |
| D2 | USN Journal 增量更新 + 索引落盘缓存 | 冷启动一次全量，此后启动秒开、实时保鲜 |
| D3 | 清理智能 = 声明式规则包 + 启发式评分器，零 LLM | 离线可用、结果可解释、行为可审计、可热更新规则 |
| D4 | 通用启发式为主，Adapter 适配为辅 | 覆盖长尾软件环境；主流应用获得更精细的分类与展示 |
| D5 | 一切删除默认可撤销（回收站或自建隔离区） | 清理软件的信任来自"从不误删" |
| D6 | 核心引擎原生语言（建议 Rust），UI 分离 | 索引百万级文件需要精确内存控制与多线程性能 |

## 5. 非目标（Non-Goals，v1）

- 注册表"优化"/系统加速类伪功能
- 驱动级文件粉碎、磁盘碎片整理
- macOS/Linux 支持
- 云端账户体系；规则包更新仅为静态签名文件下载

## 6. 成功指标（验收）

| 指标 | 目标 |
|------|------|
| 全卷索引（100 万文件，NVMe，冷） | < 10 s |
| 索引热启动（缓存 + USN 追赶） | < 1 s |
| Top-N 大文件/大文件夹查询 | < 100 ms |
| 索引内存占用（400 万文件） | < 300 MB |
| 误删（用户申诉且非用户确认项） | 0 容忍，隔离区兜底 |
| 典型家用机可释放空间识别率 | ≥ 主流竞品（WizTree 找大小 + CCleaner 找垃圾的并集） |
